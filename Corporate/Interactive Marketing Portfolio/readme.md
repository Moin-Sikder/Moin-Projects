# Marketing Portfolio Website

A modern, responsive portfolio website for a BBA Marketing graduate showcasing technical and marketing skills.

## Features

- **Responsive Design** - Works on all devices
- **Modern UI/UX** - Clean and professional design
- **Interactive Elements** - Smooth animations and transitions
- **Contact Form** - Functional contact form with validation
- **Project Showcase** - Display your marketing projects effectively

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Variables)
- JavaScript (ES6+)
- No external dependencies

## File Structure 

```
marketing-portfolio/
├── js/
│   └── script.js
├── styles/
│   └── main.css
├── index.html
└── README.md
```

## Setup

1. Clone or download this repository
2. Open `index.html` in your browser
3. Customize the content, colors, and images

## Customization

### Colors
Edit the CSS variables in `styles/main.css`:
```css
:root {
    --primary: #2563eb;
    --secondary: #64748b;
    --accent: #f59e0b;
}
